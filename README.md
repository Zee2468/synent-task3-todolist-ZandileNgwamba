# 📝 To-Do List Manager

## Internship Task 3
Author: Zandile Ngwamba

## Description

A command-line To-Do List Manager built using Python.

The application allows users to:

- View Tasks
- Add Tasks
- Complete Tasks
- Delete Tasks
- Search Tasks
- View Task Statistics

Tasks are stored permanently using a JSON file.

- JSON
